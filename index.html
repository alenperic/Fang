Text Fang/De-Fang Tool

To use this tool, simply start typing in the left text area. The page will automatically detect whether the entered text is in a 'fanged' format (standard URLs and indicators) or 'de-fanged' format (with hxxp and brackets). It will then auto-convert the text accordingly and display it in the right text area. For example, typing 'http://www.example.com' will convert it to 'hxxp[:]//www[.]example[.]com', while typing 'hxxp[:]//www[.]example[.]com' will revert it to 'http://www.example.com'.
